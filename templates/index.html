What is groupvent?

Aggregated Search

Search for anything you want in any area. Type in a keyword, a type of cuisine, or the name of a venue!

View Past Events

Trying to remember that awesome event you put together last year? Use groupvent to record all of those wondrous memories.

Invite Your Friends

Make events and invite your friends straight from the web application! No need to go to your GMail or Facebook!



groupvent was created by Jane Argodale, Edric Huang, Marlena Lui, and Dorit Rein of Pd. 6 Software Development with the user in mind. We wanted to create a user-friendly event planner that was simple to use and interact with, allowing users to search with keywords and locations.
